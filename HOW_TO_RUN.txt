Hand Gesture Control - How to run

Prerequisites:
- OS: Windows (tested)
- Python 3.10+ installed
- A working webcam

Quickest Setup:
1. Clone/download this repository
2. Open PowerShell in the project folder
3. Run these commands ONCE:
   python -m venv venv
   .\venv\Scripts\Activate.ps1
   pip install -r requirements.txt

Every Time You Want to Run the App:
   .\venv\Scripts\Activate.ps1
   python main.py

Or use the batch shortcut (if available):
   Double-click `run_app.bat`

Troubleshooting:
- If you see import errors, re-run: `pip install -r requirements.txt`
- Allow camera access when prompted by Windows
- Logs are saved to `app.log` in the project folder

Enjoy! Point your hand at the camera and follow the gestures in README.md.
